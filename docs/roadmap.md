# Vyauma Project Roadmap

This roadmap outlines the development milestones for the Vyauma language.

---

## Phase 1 — Foundation (Current)

- Create repository structure  
- Implement CLI runner  
- Add basic syntax support (`print`)  
- Publish documentation  
- Enable CI / automated testing  
- Release v0.1.0  

---

## Phase 2 — Language Core

- Implement lexer  
- Implement parser → AST  
- Implement basic runtime  
- Add variables  
- Add arithmetic operations  
- Release v0.2.0  

---

## Phase 3 — Control Flow

- Add `if/else`  
- Add `loop` constructs  
- Add boolean evaluation  
- Release v0.3.0  

---

## Phase 4 — Functions

- Add `func` keyword  
- Implement call/return  
- Add simple standard library  
- Release v0.4.0  

---

## Phase 5 — Optimized Interpreter

- Implement bytecode  
- Add VM execution engine  
- Add multi-device compatibility layer  
- Release v0.5.0  

---

## Phase 6 — Ecosystem

- Package manager (future)  
- Language server (VS Code extension)  
- Formatter + linter  
- Official documentation site  

---

## Phase 7 — Stable Release

- Finalize syntax  
- Finalize runtime  
- Release **Vyauma v1.0**  

---

This roadmap evolves as the project grows.
